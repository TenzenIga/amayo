# Клон Reddit

Функциональный клон Reddit, созданный как пет-проект для глубокого изучения современных веб-технологий. Проект служит экспериментальной площадкой, где я тестирую новые подходы, обновляю зависимости и отрабатываю практики разработки.

Демо: https://www.amayo.ru/

## Работа с контентом

- **Создание, редактирование и удаление** постов и сообществ

- **Медиа-контент:** Загрузка изображений к постам, смена аватара и баннера сообществ

## Комментарии (расширенная система)

- **Многоуровневые комментарии:** Возможность оставлять ответы на комментарии (вложенные комментарии)

- **Полное управление:** Редактирование и удаление своих комментариев

- **Интерактивность:** Система голосования за комментарии

## Социальные взаимодействия

- **Система голосования:** Лайки/дизлайки для постов и комментариев

- **Подписки:** Возможность подписываться на интересующие сообщества

- **Индивидуальная лента:** Персонализированная лента на основе подписок пользователя

## Пользовательский опыт

- **Бесконечная прокрутка (Infinity Scroll):** Плавная загрузка контента по мере скролла

- **Адаптивный дизайн**: Оптимизация для всех устройств

- **Темы**: Поддержка светлой и темной темы

## Технологический стек

## Frontend

- **Angular** - основной фреймворк

- **NgRx** - управление состоянием приложения

- **Bootstrap** - UI компоненты и стилизация

## Backend

- **Express** - серверная платформа

- **PostgreSQL** - база данных

- **TypeORM** - ORM для работы с базой данных

## Особенности разработки

Проект постоянно эволюционирует - я регулярно обновляю Angular, добавляю новые функции и улучшаю существующий код. Недавно была настроена полноценная CI/CD система через GitHub Actions с автоматическим деплоем на VPS.

Это мой личный "полигон" для экспериментов, где я могу свободно тестировать идеи без жестких архитектурных ограничений.
